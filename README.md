# 人生重开模拟器 (Life Restart)

简体中文 | [English](README_EN.md)

## 游戏介绍

一款基于原生 HTML/CSS/JavaScript 开发的人生模拟网页游戏。你将扮演一个角色，从婴儿期到老年期，经历各种人生事件，做出选择，改变命运。

## 游戏特色

- 🎮 **完整人生历程** - 从出生到100岁，经历婴儿期、童年期、少年期、青年期、中年期、老年期
- 🎯 **属性系统** - 智力、体质、魅力、运气四大属性影响人生走向
- 🎲 **随机事件** - 100+精心设计的随机事件，每次游戏体验都不同
- 🏆 **多样结局** - 18种不同结局类型，传奇人生、人生赢家、富甲一方等
- 📱 **响应式设计** - 支持桌面端、平板和移动设备
- 💾 **存档系统** - 支持多存档槽位本地存档
- ⭐ **天赋系统** - 随机天赋选择，影响初始属性
- 🎖️ **成就系统** - 30+成就等待解锁
- 📊 **统计系统** - 记录游戏数据，追踪人生轨迹

## 如何运行

### 方法1：直接打开

在浏览器中直接打开 `index.html` 文件即可开始游戏。

### 方法2：本地服务器

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx http-server

# 使用 PHP
php -S localhost:8080
```

然后在浏览器中访问 `http://localhost:8080`

## 游戏玩法

1. **创建角色** - 输入名字、选择性别、分配属性点
2. **选择星座** - 不同星座有属性加成
3. **选择天赋** - 从随机天赋中选择1-3个
4. **经历人生** - 每年会触发随机事件，做出你的选择
5. **自动/手动** - 支持自动播放和手动控制
6. **人生总结** - 百年后查看你的成就和评价
