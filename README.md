# 📊 Picture Can See

一个强大的数据可视化工具，支持多种图表类型和交互式展示。

## ✨ 功能特性

- 📈 支持多种图表：柱状图、折线图、饼图、散点图等
- 🎨 丰富的主题和配色方案
- 📱 响应式设计，支持移动端
- 💾 支持数据导入导出
- 🔗 支持图表分享

## 🚀 快速开始

### 方式一：直接使用

打开 `index.html` 即可使用。

### 方式二：本地开发

```bash
# 克隆仓库
git clone https://github.com/badhope/picture-can-see.git
cd picture-can-see

# 启动本地服务器
python -m http.server 8000
# 或
npx serve
```

访问 http://localhost:8000

## 🛠️ 技术栈

- HTML5 / CSS3 / JavaScript
- Chart.js / ECharts
- Tailwind CSS

## 📝 使用指南

1. 准备数据（CSV/JSON 格式）
2. 在页面中选择图表类型
3. 上传或粘贴数据
4. 自定义样式和配置
5. 导出图表或分享链接
