# Desktop Assets

This directory contains application assets for the desktop version.

## Required Files

- `icon.png` - Application icon (512x512 recommended)
- `icon.ico` - Windows icon (256x256)
- `icon.icns` - macOS icon

## Creating Icons

You can use tools like:
- [electron-icon-builder](https://github.com/nickolashkraus/electron-icon-builder)
- [icon-gen](https://github.com/akabekobeko/npm-icon-gen)

Or create manually with image editors like GIMP, Photoshop, or online tools.
