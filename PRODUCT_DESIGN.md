# Picture Can See - 产品设计计划书

## 一、产品概述

### 1.1 产品定位
**Picture Can See** 是一款本地优先的数据可视化工具，支持拖拽导入数据、智能图表推荐、AI辅助分析。

### 1.2 产品形态
| 版本 | 定位 | 功能范围 | 技术栈 |
|------|------|----------|--------|
| **网页版** | 引流入口，阉割版 | 基础图表生成、图片导出、功能介绍、下载引导 | HTML + CSS + JS + D3.js |
| **桌面版** | 完整功能 | 全部功能 | C#/.NET + WinUI 3 + ScottPlot |

### 1.3 目标用户
- 个人用户/小白：简单易用，快速出图
- 数据分析师：功能专业，界面简洁

### 1.4 核心使用场景
- ✅ 快速出图（办公）
- ✅ 学术/报告图表
- ✅ 个人探索
- ✅ 商业展示

| 数据库 | 连接方式 |

## 四、可视化层设计

### 4.1 支持的图表类型

#### 基础图表
| 图表类型 | 适用场景 | 数据要求 |
|----------|----------|----------|
| 柱状图 | 分类对比 | 1分类 + 1数值 |
| 折线图 | 趋势分析 | 1时间/分类 + 1数值 |
| 饼图 | 占比展示 | 1分类 + 1数值 |
| 散点图 | 相关性分析 | 2数值 |
| 面积图 | 累积趋势 | 1时间/分类 + 1数值 |
| 雷达图 | 多维对比 | 1分类 + 多数值 |

#### 扩展图表（桌面版）
| 图表类型 | 适用场景 |
|----------|----------|
| 热力图 | 密度分布 |
| 树图 | 层级结构 |
| 桑基图 | 流向关系 |
| 漏斗图 | 转化分析 |
| 仪表盘 | KPI展示 |
| 地图 | 地理数据 |

### 4.2 智能图表推荐

```
推荐逻辑：
1. 规则匹配（优先）
   ├── 1个分类 + 1个数值 → 柱状图/饼图
   ├── 1个时间 + 1个数值 → 折线图
   ├── 2个数值 → 散点图
   ├── 多个数值 → 雷达图
   └── 地理数据 → 地图

2. AI智能推荐（补充）
   └── 分析数据特征，给出推荐理由

3. 用户选择
   └── 显示推荐列表，用户选择
```

### 4.3 图表配置选项

| 配置项 | 选项 |
|--------|------|
| 标题 | 主标题、副标题、位置、字体大小 |
| 坐标轴 | X/Y轴标题、显示/隐藏、轴线 |
| 颜色 | 6种配色方案、自定义颜色 |
| 动画 | 类型、时长、缓动函数 |
| 图例 | 位置、布局 |
| 交互 | 提示框、缩放、平移 |

### 4.4 图表交互功能

- ✅ 悬停提示
- ✅ 点击筛选
- ✅ 缩放平移
- ✅ 动画效果

### 4.5 图表标注功能

- ✅ 文字标注

### 4.6 图表管理
